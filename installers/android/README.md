# BONZO_media_HUB — Android local setup

## Wymagania
- Android + Chrome
- (Opcjonalnie) Termux + storage access (`termux-setup-storage`)

## Szybki setup (Termux)
1. Skopiuj `setup-bonzo-android.sh` do Termux
2. Uruchom:
   - `chmod +x setup-bonzo-android.sh`
   - `./setup-bonzo-android.sh`
3. Otwórz BONZO_media_HUB i przejdź do `INSTALL` → `LOCAL_SETUP`
4. Użyj `CONNECT_MUSIC_FOLDER` i `CONNECT_VIDEO_FOLDER`

## Bez Termux
Możesz ręcznie utworzyć:
- `BONZO_media_HUB_Local/Music`
- `BONZO_media_HUB_Local/Movies`
i podłączyć je z poziomu LOCAL_SETUP w aplikacji.
